# Fetch Weather CLI

A Python-based Command Line Interface (CLI) tool that fetches real-time weather data using the OpenWeatherMap API.

## Features
* **Real-time Updates:** Get current temperature, humidity, and wind speed for any city.
* **Data Logging:** Option to save weather reports to a local `weather_data.json` file.
* **Metric Units:** Displays temperature in Celsius.

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install requests`
3. Run the script: `python weather.py`

## Tech Stack
* Python 3
* OpenWeatherMap API
* JSON for data persistence
